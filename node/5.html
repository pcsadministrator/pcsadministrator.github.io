Platinum Code Shipping Policy

Platinum Code Solutions, LLC is a provider of software services and products. Most products are delivered electronically. Shipping policies apply only to tangible products.
The purchaser must pre-pay shipping costs including appropriate insurance for any defective item returned to Platinum Code Solutions, LLC. Platinum Code Solutions, LLC shall not be liable for risk of loss or damage during shipment of your returned items if you fail to insure the shipment.
Platinum Code Solutions, LLC will ship the repaired or replacement product(s) to Purchaser via Ground Service (freight prepaid) if you use an address in the continental United States. For shipments to other locations, Purchaser must pre-pay any shipping charges, insurance, export taxes, custom duties and taxes including VAT taxes, or any other charges associated with transportation of your Platinum Code Solutions, LLC. Purchaser assumes the risk of loss. Platinum Code Solutions, LLC. shall not be responsible for failure of the delivery service to make on-time delivery. If Purchaser requests a shipping method other than Ground Service, Purchaser must pre-pay the difference in cost before Platinum Code Solutions, LLC. will ship the replacement product.
NOTE: Before you ship the product(s) to Platinum Code Solutions, LLC, back up any data on any storage device(s) in the product(s). Platinum Code Solutions, LLC does not accept liability for lost data or software.
